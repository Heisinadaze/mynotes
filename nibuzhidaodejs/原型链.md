
* 所有的引用类型都有一个`__proto__`属性

* 所有函数都拥有`prototype`属性

# 从构造函数开始

```

function Word (words) {
  this.words = words
}

Word.prototype = {
  alert () {
    console.log(this.words) // hello world
  }
}

var w = new Word('hello world');
w.print = function () {
  console.log(this.words) // hello world
  console.log(this) // { words: 'hello world', print: [Function] }
}

w.print()
w.alert()

```

```
w.__proto__ === Word.prototype
```
