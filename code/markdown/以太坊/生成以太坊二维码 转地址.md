### use

* 生成二维码 (base64格式)

> 扫出来的格式大致： 'ethereum:0x7cB57B5A97eAbe94205C07890BE4c1aD31E486A8

```
import EthereumQRPlugin from 'ethereum-qr-code'

        const qr = new EthereumQRPlugin()
        qr.toDataUrl({
          to: this.info.name2
        }, {
          selector: '#my-qr-code'
        }).then(res => {
          this.imgUrl = res.dataURL
        })
```
