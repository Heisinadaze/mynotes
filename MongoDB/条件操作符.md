
# 条件操作符

```
$gt -------- greater than  >

$gte --------- gt equal  >=

$lt -------- less than  <

$lte --------- lt equal  <=

$ne ----------- not equal  !=

$eq  --------  equal  =
```


eg:
```
db.col.find({
  likes: {$lt: 200, $gt: 100}
})
```
