
### Limit(n)

读取指定数量的数据记录,接受一个数字参数。
`db.col.find().limit(Number)`

### Skip(n)

使用`skip()`跳过指定数量的数据

### sort()

通过参数指定排序的字段，1为升序， -1为降序。

```
db.col.find().sort({KEY:1})
```

### 索引
`ensureIndex()`创建索引

```
db.col.ensureIndex({KEY:1})

1为升序， -1为降序。
```


eg:
```
复合索引

db.col.ensureIndex({"title":1, "likes":-1})
```

## 覆盖索引查询

* 所有的查询字段是索引的一部分

* 所有的查询返回字段在同一个索引中
