

### 元素选择器

```
$('元素名')

$('.class')

$('#id')

```



### 属性选择器

```
$('[href]')

$('[href="#"]')

$('[href!="#"]')

$('[href$=".jpg"]') // 选取所有 href 值以 ".jpg" 结尾的元素。

```
