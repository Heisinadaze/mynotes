> column:3;表示均分三列

```
column-width     设置列宽（建议使用em为单位）,不加单位表示均分几列，加单位表示一列有多少字  

column //设置列数

columns: 5em 5; // 复合属性（优先选择较少列数的进行使用）

column-gap // 列缝隙之间的宽度

column-rule // 设置缝隙中的线的样式

* column-rule-style
* column-rule-color
* column-rule-width

column-span // 跨列（FF模式不支持）



    -moz-column
    -webkit-column
    -ms-column
    -o-column
    column
