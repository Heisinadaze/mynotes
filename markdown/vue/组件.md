
### prop验证

```
props: {
  propA: Number,
  propB: [String, Number],
  propC: {
    type: String,
    require: true
  },
  propD: {
    type: Number,
    default: 100
  },
  propE: { // 自定义验证函数
    validator (value) {
      return value > 10
    }
  }
}

```
> type: String, Number, Boolean, Function, Object, Array, Symbol
> type也可以是自定义构造器函数，使用`instanceof`
